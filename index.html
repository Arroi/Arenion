Arenion

Open-source Webhook controller.

Don't believe it? try it yourself!

Simply type whatever in the input, and submit it!

it doesn't require json, how awesome!

How it works:

It uses Javascript to ship the input values to your webhook using an embed.
To modify it, go to arenion/site_controller/result.js.

To modify the webhook link: 

const webhookUrl = "https://discord.com/api/webhooks/WEBHOOK_ID/WEBHOOK_TOKEN";
this tutorial assumes that you already know how to work with webhooks, replace the existing webhook to your
webhook link.
To modify the embed: 

const webhookMessage = {
embeds: [
{
title: "Input has been sent.", // The title of the embed
description: "", // The description of the embed
color: 0xff0000, // the color of the embed
 fields: [ // here are the field of the embed.
{
 name: "Message content:", // the title of the embed message
value: "", // values that has been shipped using the input.
inline: true, // recommended to keep it true.
},
],
},
],
};
To modify the embed, you can change it's title as shown above, for the sake of this tutorial, it has set to "Input has been sent" You could modify the color & description to your liking.
You can modify the name of the message. but it's recommended to not modify the values as they're empty and should not consider modifying them.

You get the idea, you can modify everything else. I'll make a tutorial about it soon. check out my channel for updates : https://www.youtube.com/channel/@itsyoboiarroi for more information!
